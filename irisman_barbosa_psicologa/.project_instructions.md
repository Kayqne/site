# Irisman Barbosa - Psicóloga Website

## Project Overview
Modern, elegant psychology practice website for Irisman Barbosa (CRP 09/18965). Professional NextJS application with full backend functionality, blog system, appointment booking, and admin dashboard.

## Design System

### Color Palette
- **Primary Gold**: `#c9a84c` (logo-matched golden accent)
- **Gold Light**: `#e2c97e` (highlights and hover states)
- **Gold Dim**: `#8a6e2f` (borders and subtle elements)
- **Background Dark**: `#0c0b09` (main background)
- **Background Dark 2**: `#111009` (section alternating background)
- **Cream**: `#f2ead8` (primary text color)
- **Cream Dim**: `#a89f8c` (secondary text)
- **White**: `#faf8f3` (headlines and important text)

### Typography
- **Headings**: 'Cormorant Garamond' (serif) - elegant, professional
- **Body**: 'Jost' (sans-serif) - clean, readable
- **Font Weights**: 300 (light), 400 (regular), 500 (medium)

### Visual Effects
- **Glow Animations**: Subtle golden glow on hover for buttons and important elements
- **Noise Overlay**: Subtle texture overlay for depth
- **Border Style**: Golden borders with low opacity (rgba(201,168,76,0.18))
- **Shadows**: Soft shadows for depth and elevation
- **Transitions**: Smooth 0.3-0.5s transitions on all interactive elements

### Animation Principles
- Soft, elegant animations (no harsh movements)
- Scroll-triggered reveals with stagger delays
- Golden glow effects on hover (box-shadow with golden color)
- Fade-up animations for sections
- Smooth parallax effects where appropriate

## Technical Architecture

### Database Schema (Prisma)
- **User**: Admin authentication (includes john@doe.com test account + irisman@barbosa.com admin)
- **BlogPost**: Blog CMS with full CRUD (title, slug, content, excerpt, coverImage, published, category, tags)
- **Appointment**: Booking system (name, email, phone, modality, message, date, status)
- **Contact**: Contact form submissions (name, email, phone, message, status)

### Authentication
- NextAuth.js v4 with Prisma adapter
- Email/password authentication
- Test Account: john@doe.com / johndoe123
- Admin Account: irisman@barbosa.com / irisman2025
- Protected admin routes

### Email Notifications
- Contact form submissions → NOTIF_ID_NOVA_MENSAGEM_DE_CONTATO
- Appointment bookings → NOTIF_ID_NOVA_SOLICITAO_DE_AGENDAMENTO
- HTML-formatted emails to admin

## Page Structure

### Public Pages
1. **Home** (`/`) - Hero with logo, intro, services preview, blog preview, CTA
2. **About** (`/sobre`) - Professional background, approach, credentials (CRP 09/18965)
3. **Services** (`/servicos`) - Therapy services offered (grid layout with numbers)
4. **Blog** (`/blog`) - Blog listing page
5. **Blog Post** (`/blog/[slug]`) - Individual blog post
6. **Contact** (`/contato`) - Contact form with WhatsApp integration
7. **Booking** (`/agendar`) - Appointment booking form

### Admin Pages (Protected)
1. **Dashboard** (`/admin`) - Overview stats
2. **Blog Management** (`/admin/blog`) - Create, edit, delete posts
3. **Appointments** (`/admin/agendamentos`) - View and manage appointments
4. **Contact Submissions** (`/admin/contatos`) - View contact messages

### API Routes
- `/api/auth/[...nextauth]` - NextAuth authentication
- `/api/signup` - User registration
- `/api/contact` - Contact form submission
- `/api/appointments` - Appointment booking
- `/api/blog` - Blog CRUD operations
